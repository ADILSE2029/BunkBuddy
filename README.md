# Bunk Buddy

A command-line attendance tracker built with Python. Tracks your class 
attendance per subject and tells you exactly how many classes you can 
skip before dropping below the 75% minimum requirement.

## What it does

Marks attendance as present, absent, or canceled for each subject. 
Calculates your current attendance percentage and warns you when you 
drop below 75%. The Bunk-o-Meter calculates how many more classes you 
can skip while staying above the limit.

## Features

- Mark attendance per subject (Present / Absent / Canceled)
- View attendance table with totals and percentages
- Warnings for subjects below 75% attendance
- Bunk-o-Meter — calculates remaining skippable classes per subject
- Reset all data

## Subjects tracked

Probability & Statistics, Technical & Business Writing, Digital Logic Design,
Introduction to Software Engineering, Object Oriented Programming, Discrete Structures

## How to run

Make sure you have Python installed, then run:

python main.py

## Files

- module1.py — contains the Bunkbuddy class with all core logic
- main.py — entry point and menu handling
- bunkbuddy — auto-generated file where attendance data is stored

## Built with

Python — file I/O, OOP, CLI input handling
